# GTIXT — Brand Guidelines v1

**Official lockup:** `GTIXT — The Global Prop Trading Index`  
**Alt (more institutional):** `GTIXT — Institutional Prop Trading Benchmark`

## Direction B — “T‑Index Monogram”
A single mark that reads as:
- **T** (Index “T”)
- **Rising bars** (benchmark / performance index)
- Works at 16px (favicons) and scales to decks.

### Clearspace
Minimum clearspace: **0.5×** the icon corner radius.

### Don’ts
No 3D, no bevel, no shadows baked into the logo, no stretching.

## Color palette
- Background: `#0B1220`
- Panel: `#0F1A2E`
- Text: `#E6EEF7`
- Accent: `#2F6BFF`
- Accent hover: `#2453CC`
- Optional data accent: `#15B8A6`

## Typography (recommended)
- Headings: **Sora** (OFL-1.1)
- Body/UI: **Inter** (SIL OFL)

## Accessibility targets
Aim for WCAG contrast thresholds:
- normal text **≥ 4.5:1**
- large/bold text **≥ 3:1**
